
# 💻 Laptop Price Pridiction Model

## Description
The Laptop Price Prediction Model is a Machine Learning project developed to predict the price of a laptop based on its specifications and hardware features. The model is trained using a dataset containing various laptop configurations such as company name, processor type, RAM size, storage capacity, GPU, operating system, screen size, and other technical details.

The main objective of this project is to understand the complete workflow of a Machine Learning regression project, starting from data preprocessing to model training and prediction.

This project is useful for:

- Students learning Machine Learning

- Beginners exploring Data Science

- E-commerce price analysis

- Understanding regression algorithms



## Output of the Project

The model takes laptop specifications as input and predicts the estimated laptop price

### Example Input:

- Brand: Dell
- RAM: 8GB
- Processor: Intel Core i5
- SSD: 512GB
- GPU: NVIDIA
- Screen Size: 15.6 inches

### Example Output:
 
- Predicted Price: ₹65,000

### The output can be displayed:

- In the terminal
- Through a GUI application
- On a web page using Flask or Streamlit
